== Installation ==

Installing this plugin as regular wordpress plugin. First you need to install WPBakery Page Builder plugin and after that install WPBakery IT Consulting Addons.

This addon plugin works like other wpbakery page builder addon plugin. Go to new page then click on WPBakery Page builder then click on Add Element you should see "IT Consult Addons" on Menu Top, click on it you will see shortcode item list.
